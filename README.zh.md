# SuperMap Mapbox

超图改版的Mapbox GL（v1.12.0）以及iclent-mapboxgl（v11.0.1）

- iclient-mapboxgl-es6.min.js
- iclient-mapboxgl.min.css
- mapbox-gl-enhance.css
- mapbox-gl-enhance.js
- mapbox-gl-enhance.mjs


## 支持多种坐标系

- EPSG4326
- EPSG4490
- EPSG4214
- EPSG4610
- EPSG3857
- ...
